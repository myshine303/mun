アニバーサリーポイント ～特別なサービス～

ご指定の月にお得にポイントがたまる「アニバーサリーポイント」で上手にポイントアップ

入会時にご指定いただいたアニバーサリー月（記念月）にショッピングをご利用いただくと、獲得ポイントが優遇される特別なサービスです。

  • プラチナ、ゴールドプレステージ会員様：2.0倍
  • ゴールド会員様：1.5倍
  • 一般、イニシャルカード、ビジネスカード会員様は本サービスをご利用いただけません。

サービス内容

入会時にご指定いただいたアニバーサリー月にショッピングをご利用いただくと、お客様のカードに応じた優遇率分のアニバーサリーポイントが基本ポイントが付与された翌月に付きます。

  • ご入会時にご指定のない場合は、本会員様の誕生月をアニバーサリー月と設定させていただきます。

プラチナ、ゴールドプレステージ会員様

アニバーサリー月の優遇率 たとえば1万円のショッピングご利用で
2.0倍 基本ポイント10ポイント＋
アニバーサリーポイント10ポイント獲得

ゴールド会員様

アニバーサリー月の優遇率 たとえば1万円のショッピングご利用で
1.5倍 基本ポイント10ポイント＋
アニバーサリーポイント5ポイント獲得
  • ショッピングご利用1,000円につきプラチナ、ゴールドプレステージ会員様は2ポイント、ゴールド会員様は1.5ポイント付与いたします。

ポイントの付き方

アニバーサリーポイントが付くタイミングは、ショッピングご利用日およびそのご請求月により異なります。

アニバーサリー月1日～15日までのご利用日分のポイントの付き方

アニバーサリー月16日～末日までのご利用日分のポイントの付き方

ご確認方法

WEB

パソコン

MUFGカードWEBサービスにログインのうえ、ご確認ください。

郵送

ご利用明細書

毎月のご利用明細書でポイントをご確認いただけます。

ご注意事項

  • アニバーサリー月の1日～末日までのご利用を対象とします。
  • ご請求月に付与する基本ポイントの翌月に、アニバーサリーポイントとして付与いたします。
  • 獲得したポイントは、MUFGカードWEBサービス、ご利用明細書ポイント表示欄の「アニバーサリー」へ表示いたします。
  • ショッピングのご利用をされた加盟店からの売上票到着時期により、ご請求月および付与月がずれる場合がございます。
  • アニバーサリー月の変更は、対象カード1枚につき1回のみ可能です。
    変更をご希望の場合、MUFGカード・アメリカン・エキスプレス・カードデスクまでお申し出ください。
  • カードの切替え（ゴールドからゴールドプレステージなどへの会員種類の変更等）が行われた場合もアニバーサリー月および変更回数制限は切替え後のカードに引き継がれます。
  • アニバーサリー月ご利用後に、アニバーサリー月変更をご希望の場合、ご利用分のご請求月14日（休日の場合は翌営業日）以前に変更されると、アニバーサリーポイントを付与しない場合がございますので、ご注意ください。

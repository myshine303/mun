グローバルPLUS

使うほどにポイント優遇率があがる「グローバルPLUS」でポイントアップ

ショッピングご利用金額に応じて、獲得ポイントを優遇する特別なサービスです。

サービス内容

プラチナ、ゴールドプレステージ、ゴールド会員様

1年間のショッピングのご利用金額に応じて、翌1年間、獲得ポイントを優遇いたします。

 

ステージ

1年間のショッピング
ご利用金額

100万円以上

50万円以上100万円未満

優遇内容

基本ポイントの50％分を加算いたします。

基本ポイントの20％分を加算いたします。

1年間のご利用金額が…
  • 入会初年度の集計期間はご入会日により13ヵ月間とする場合がございます。
  • ご利用金額はカードごとの集計となります。MUFGカード・アメリカン・エキスプレス・カードを複数枚お持ちの場合でも合算されません。

一般会員様

1年間のショッピングのご利用金額に応じて、翌1年間、獲得ポイントを優遇いたします。

 

ステージ

1年間のショッピング
ご利用金額

100万円以上

50万円以上100万円未満

優遇内容

基本ポイントの20％分を加算いたします。

基本ポイントの10％分を加算いたします。

1年間のご利用金額が…
  • 入会初年度の集計期間はご入会日により13ヵ月間とする場合がございます。
  • ご利用金額はカードごとの集計となります。MUFGカード・アメリカン・エキスプレス・カードを複数枚お持ちの場合でも合算されません。

イニシャルカード会員様

月間（１ヵ月）のショッピングご利用金額に応じて当月の獲得ポイントを優遇するサービスです。

 

ステージ

1ヵ月のショッピング
ご利用金額

10万円以上

3万円以上10万円未満

優遇内容

基本ポイントの50％分を加算いたします。

基本ポイントの20％分を加算いたします。

1ヵ月のご利用金額が…
  • 月間（1ヵ月）のご利用は、原則毎月16日～翌月15日を対象期間とし、翌々月ご請求分にポイント付与いたします。

ご確認方法

WEB

MUFGカードWEBサービスにログイン後、メニューの「ポイント照会・応募」を押してください。

パソコン

ご利用明細書

毎月のご利用明細書でもご確認いただけます。

ご注意事項

  • ご利用金額の集計は、当社にてカードごとに自動集計いたします。
  • 集計期間・累積金額はMUFGカードWEBサービス、ご利用明細書へ表示いたします。
  • ショッピングのご利用をされた加盟店からの売上票到着時期により、当年または当月のご利用金額に集計しない場合がございます。
  • ボーナス一括払いでのご利用金額は、ご利用時期により当年または当月のご利用金額に集計しない場合がございます。
  • グローバルPLUSとして付与するポイントは、MUFGカードWEBサービス、ご利用明細書ポイント欄の「グローバルPLUS」欄に表示いたします（小数点以下は切り捨てとなります）。
  • 集計対象は、グローバルポイント対象のご利用金額とします。
  • キャンペーンなどで付与されるポイントは、優遇対象外です。
  • デュアルスタイルの場合、MUFGカード・アメリカン・エキスプレス・カードでの国内でのショッピングご利用分がいつでも1.5倍となるため、グローバルPLUSは適用いたしません。

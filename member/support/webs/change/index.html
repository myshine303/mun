ID・パスワード変更お手続き方法

〜安全・安心のために〜

昨今、他人のID・パスワードを不正に入手し、本人になりすまして様々なサイトにログインを試みるといった行為が多発しております。弊社WEBサービスのID・パスワードを他のサイトでも兼用されている場合、思わぬ被害にあう可能性がございますので、下記の注意点をご確認のうえ、ID・パスワードの定期的な変更をお願いいたします。

  1. 複数のサイトで同一のID・パスワードを使用しない
  2. ID・パスワードは他人に教えない
  3. パスワードは英字・数字を組み合わせるなど、推測されやすいパスワードを使用しない
  4. ID・パスワードは定期的に変更する

一般社団法人日本クレジット協会のホームページでも注意喚起を行っております。ぜひご確認ください。

インターネット取引におけるID・パスワードの使いまわしによる不正使用被害にご注意ください

WEBサービスのID・パスワード変更方法についてご案内いたします。
カードブランドによってお手続き方法が異なりますので、お手持ちのカードをご確認のうえお手続きください。

  • MUFGカード（UFJカード含む）会員の方／MUFGカード・アメリカン・エキスプレス・カード会員の方
  • DCカード会員の方
  • NICOSカード会員の方

MUFGカード（UFJカード含む）会員の方／MUFGカード・アメリカン・エキスプレス・カード会員の方

  1. NEWS+PLUS（https://www2.cr.mufg.jp/newsplus）にアクセス
  2. MUFGカードWEBサービスのID・パスワードを入力し「ログイン」を押す
  3. 「WEBサービストップ」を押す
  4. メニューの「お客様情報の照会・変更」を押す
  5. IDの変更をご希望の場合は「WEBサービスIDの変更」より、パスワードの変更をご希望の場合は「WEBサービスパスワードの変更」より、画面の案内にしたがってお手続きください

DCカード会員の方

個人会員の方

  1. NEWS+PLUS（https://www2.cr.mufg.jp/newsplus）にアクセス
  2. DC WebサービスのID・パスワードを入力し「ログイン」を押す
  3. 「WEBサービストップ」を押す
  4. メニューの「お届け内容の変更・照会」の「各種お手続き」を押す
  5. IDの変更をご希望の場合は「ID変更」より、パスワードの変更をご希望の場合は「パスワード変更」より、画面の案内にしたがってお手続きください

法人会員の方

  1. DC Webサービス-法人カード-（https://houjin.dccard.co.jp/dccp/DCCPEntry/htmls/login/corporation.shtml）にアクセス
  2. DC WebサービスのID・パスワードを入力し「ログイン」を押す
  3. メニューの「各種変更のお手続き」を押す
  4. IDの変更をご希望の場合は「ID変更」より、パスワードの変更をご希望の場合は「パスワード変更」より、画面の案内にしたがってお手続きください

カード発行会社が提携（FC契約）金融機関各行の方

  1. NEWS+PLUSの提携（FC契約）金融機関一覧（https://www2.cr.mufg.jp/newsplus/bk_info）にアクセスいただき、ご所属の提携（FC契約）金融機関をご選択ください
  2. DC WebサービスのID・パスワードを入力し「ログイン」を押す
  3. 「WEBサービストップ」を押す
  4. IDの変更をご希望の方はメニューの「お客様情報の変更」の「ID変更」より、パスワードの変更をご希望の方はメニューの「パスワード変更」より、画面の案内にしたがってお手続きください

NICOSカード会員の方

  1. NEWS+PLUS（https://www2.cr.mufg.jp/newsplus）にアクセス
  2. Web会員サービス「Net Branch」のID・パスワードを入力し「ログイン」を押す
  3. 「WEBサービストップ」を押す
  4. メニューの「お届け内容の変更」を押す
  5. IDの変更をご希望の場合は「NICOS ID任意変更」より、パスワードの変更をご希望の場合は「パスワード変更」より、画面の案内にしたがってお手続きください

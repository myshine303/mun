ご融資書面WEB通知サービス規定

第1条（本サービスの内容）

  1. 「ご融資書面WEB通知サービス」（以下、「本サービス」といいます。）は、第2条第2項に定める本サービス利用者に対し、次項に定める書面（以下、「法定書面等」といいます。）を、郵送による方法に代えて、当社所定のローンカードWEBサービス利用者規定（以下、「原規約」といいます。）に定めるサービス（以下、「WEBサービス」といいます。）の一環として、本規定に従い電磁的方法により提供するサービス（以下、書面に記載されている事項であって電磁的方法により提供される情報を「電子書面」といいます。）のことをいいます。

  2. 本サービスの対象となる書面は、次の書面とします。

    1. お客様が融資を利用した都度送付する「ご融資明細書」（貸金業法第17条第1項に定める書面）
    2. ご利用代金請求書
    3. 次項に定める方法により告知された書面
  3. 当社は、当社のホームページその他の当社所定のWEBサイト（以下、「WEBサイト」といいます。）上で告知することにより、本サービスの対象となる書面を変更または追加することができるものとします。
  4. 本規定における「当社」とは、原規約において定める意義を有するものとします。

第2条（本サービスの利用者）

  1. WEBサービスの利用者であって、本サービスの利用を希望する者は、本規定を承諾したうえ、当社所定の方法により本サービスの利用登録を行うものとし、当社が適当と認めた場合に、本サービスを利用することができるものとします。なお、本サービスの利用登録においては、当社に対して自己が常用するe-mailアドレスの届出を行うことを条件とし、当該届出が行われない場合には利用登録はできないものとします。
  2. 本サービスの利用登録を行い、当社が適当と認めたWEBサービスの利用者を、本規定において「本サービス利用者」といいます。
  3. 本サービス利用者は、パソコン及びスマートフォンその他これらに類する電子機器等（以下「パソコン等」といいます。ただし、携帯電話及びPHSは除きます。）によりインターネット接続できる環境及び電子書面をパソコン等に備えられたファイルに記録し、この記録を出力して紙媒体に印刷することができる環境を有していなければならないものとします。
  4. 当社は、本サービスの利用登録に係る本サービス利用者の個人情報については、原規約の定めに従って取扱うほか、本サービスの履行及び本サービスに関する利用案内その他の本サービス利用者に対する情報提供のために利用することができるものとし、本サービス利用者はこれに同意するものとします。

第3条（書面の提供方法）

  1. 当社は、WEBサイトに本サービス利用者宛の電子書面を次項に定めるファイルへの記録方式で保存し、本サービス利用者が原規約に定める手続に従いWEBサービスにアクセスする方法により、当該電子書面を閲覧できるようにします。
  2. ファイルへの記録の方式はAdobe Reader（Ver.6.0以上）のソフトウェアを使用するものとします。
  3. 本サービス利用者は、第1項により提供された電子書面を閲覧し、自らのパソコン等にダウンロードのうえ保存するものとします。

  4. 本サービスの利用期間中は、原則として当社から本サービス利用者への法定書面等の郵送は行いません。また、第1項及び第3項の方法により提供された電子書面については、本サービスの利用期間の終了後においても、本サービス利用者に対し法定書面等の郵送は行いません。ただし、次のいずれかに該当する場合は、当社は、法定書面等を本サービス利用者宛に郵送することができるものとします。

    1. 法令等により法定書面等の郵送が必要とされる場合
    2. 通信上のトラブル、インターネット環境の不具合等により、本サービス利用者において電子書面の内容が確認できないと当社が認めた場合
    3. 次条により本サービス利用者が本サービスを利用することができない場合
    4. その他当社が法定書面等の郵送を必要と判断した場合

第4条（電子書面の閲覧・保存等不可時の対応）

本サービス利用者は、前条第3項に定める電子書面の閲覧、ダウンロードまたは保存のいずれかができなかった場合は、当社にその旨を申出るものとします。この場合は、本サービス利用者は、本サービスを利用することはできないものとします。

第5条（登録事項の変更）

本サービス利用者は、e-mailアドレスその他本サービスの利用登録に係る事項の変更を行った場合は、遅滞なくWEBサービスの所定のメニューを利用して、当社に対して当該変更内容の届出を行うものとします。

第6条（本サービスの利用に必要なOS・ソフトウェアの種類等）

本サービスの利用に必要なOS、WEB閲覧用ソフトウェア（ブラウザ）及びPDFファイル閲覧用ソフト等（以下、「必要OS等」といいます。）の種類及びそのバージョンは、当社が任意に指定することができるものとし、当社は、当該指定した内容をWEBサイト上に掲示します。なお、必要OS等は、本サービス利用者の負担及び責任において準備するものとします。

第7条（本サービスの利用の中止等）

  1. 本サービス利用者が本サービスの利用の中止を希望する場合は、当社所定の方法により本サービスの利用登録を解除する旨の届出を行うものとします。

  2. 当社は、本サービス利用者が次の各号のいずれかに該当する場合、当該本サービス利用者に通知することなく本サービスの利用登録を解除することができるものとします。

    1. WEBサービスの利用者でなくなった場合またはWEBサービスの利用を一時停止された場合
    2. 当社が発行するカードの会員としての資格を喪失した場合
    3. 本規定に違反したと当社が判断した場合
    4. 必要OS等を有していないことが判明した場合
    5. その他本サービスの利用が不適当と当社が判断した場合
  3. 本サービス利用者は、前二項に基づき本サービスの利用登録が解除された時点以降は、本サービスの利用ができなくなるものとします。

第8条（本サービスの終了・中断・変更等）

  1. 当社は、事前の予告なしに、営業上その他の理由により本サービスを終了もしくは中断し、または内容を変更することができるものとします。この場合、その旨をWEBサイト上で告知するものとします。
  2. 当社がWEBサービスを中止または廃止した場合には、本サービスは、当然に終了するものとします。
  3. 本サービス利用者は、本サービスの内容は日本国の法律のもとに規制されることがあることを承諾するものとします。

第9条（免責事項）

  1. 本サービス利用者は、通信上のトラブル、インターネット環境の不具合その他やむを得ない事由により、本サービスの利用ができず、あるいは本サービスの提供が遅延する場合があることをあらかじめ承諾するものとします。
  2. 当社は、当社の故意または重過失に起因する場合を除き、本サービスの利用に関して本サービス利用者に生じた不利益及び損害については一切責任を負わないものとします。
  3. 当社は、前条に規定された本サービスの終了、中断または変更等に関して、本サービス利用者に生じる不利益及び損害について、一切責任を負わないものとします。

第10条（本規定の変更等）

  1. 当社は、本サービス利用者への事前通知または承諾なくして、WEBサイトへの掲載またはe-mail等で通知するなどの所定の方法により本規定を随時変更することができるものとします。
  2. 本サービス利用者は、前項の掲載または通知時点において当該変更に同意したものとします。
  3. 本サービスの利用に際し、本規定に定めのない事項は原規約を適用するものとします。
